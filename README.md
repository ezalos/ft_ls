### :gear: Makefile
-----------

Run `make` command in order to compile the project and dependencies. 
Run `make <rule>` in order to use a specific rule.

**Rules** 

| **basic**     |      **descriptions**                 |
|:-------------:|:-------------------------------------:|
| clean         | delete obj files and folders          |
| fclean        | clean all objs and remove executable  |
| re            | fclean + make                         |
| rere          |   <>                                  |
